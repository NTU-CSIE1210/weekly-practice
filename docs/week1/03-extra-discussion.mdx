# Extra Discussion Questions

1. **(Reverse an Array)** Given an array $A[0..n-1]$, please implement `Reverse(A)` that reverses $A$. 
2. **(Rotate an Array)** Given an array $A[0..n-1]$, how would you implement a rotation to this array?
Namely, `RotateLeft(A, k)` should rearrange $A$ into the following order: $A[k], A[k+1], \ldots, A[n-1], A[0], \ldots, A[k-1]$.
