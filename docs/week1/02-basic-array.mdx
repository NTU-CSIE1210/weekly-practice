## Two Dimensional Array

<Tabs groupId="programming-language">
  <TabItem value="java" label="Java">

Java's 2D array looks more like "array of 1D arrays".
Each array can be initialized to different sizes.

```java
public class array1 {
  public static void main(String[] args) {
    int[][] a;
    a = new int[100][];
    for (int i = 0; i < 100; i++) {
      a[i] = new int[i];
      for (int j = 0; j < i; j++) {
        a[i][j] = i + j;
      }
      System.out.println(a[i].length);
    }
  }
}
```
