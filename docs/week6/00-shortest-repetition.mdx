# {frontMatter.title}

:::info[The Task]

You are given a string $S$. Find the shortest string $X$ such that $S$ is a repetition of $X$.

:::

For example, if $S=\texttt{abcabcabcabc}$ then your algorithm should output $X=\texttt{abc}$.

## Goals

* Find an $O(n \sqrt{n})$ time algorithm. (Hint: enumerate all possible lengths of $X$.)
* Find an $O(n \log n)$ time algorithm. (Hint: built on above idea, but check each match in $O(1)$ time using Hashing.)
* Find an $O(n)$ time algorithm. (Hint: ...)
